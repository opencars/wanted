## Development

Build the binary

```sh
make
```

Start postgres

```sh
docker-compose up -Vd postgres
```

Run sql migrations

```sh
migrate -source file://migrations -database postgres://postgres:@127.0.0.1/wanted\?sslmode=disable up
```

Run the web server

```sh
./bin/server
```
